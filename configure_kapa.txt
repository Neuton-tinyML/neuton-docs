This repository contains documents used to train an AI Assistant for the lab.neuton.ai platform using kapa.ai as a service.

Main information is provided in the urls_for_scraping.txt. These links were added one by one into kapa.ai as webcrawl data sources and configured to use specific parts of web pages with specyfying css markers.

Additional info, which is not presented on the neuton documentation web-version are provided in markdown files to which are automatically pulled by kapa.ai from this repo.

fac.csv has been used to manually insert each question-answer pair into kapa.ai custom answers data source.
